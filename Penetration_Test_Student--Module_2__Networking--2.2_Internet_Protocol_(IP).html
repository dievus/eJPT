2.2 Internet Protocol (IP)


Internet Protocol


• Protocol that runs on the Internet layer of TCP/IP
• In charge of delivering datagrams (IP packets) to hosts involved in commuication, and uses IP addresses to identify hosts
• Majority of networks run IPv4
◇ Consists of four octets, each four bytes in length; total IPv4 is 32 bits
• Reserved IPv4 addresses include:
◇ 0.0.0.0 - 0.255.255.255 - represents “this” network
◇ 127.0.0.1 - 127.255.255.255 - represents the local host
◇ 192.168.0.0 - 192.168.255.255 - reserved for private networks

IP/Mask


• Identification of a host includes needing to know the network. Understanding this requires a netmask, or subnet mask
• With an IP/netmask pair, you can identify the network part and the host part of an IP address
◇ IP - 192.168.5.100
◇ Subnet - 255.255.255.0
• In order to determine the network part, a bitwise AND operation has to be conducted between the netmask and IP
