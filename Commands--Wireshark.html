Wireshark

• Basic Capture filters include:
◇ ip
▪ Only packets using IP as layer 3 protocol
◇ not ip
▪ Packets not using IP
◇ tcp port 80
▪ Packets where the source or destination is TCP port 80
◇ net 192.168.1.0/24
▪ Packets from and to the specified network
◇ src port 1234
▪ Source port must be 1234; transport protcol does not matter
◇ src net 192.168.1.0/24
▪ Source IP address must be in the specified network
◇ host 192.168.45.65
▪ All the packets from or to the specified host
◇ host www.examplehost.com
▪ All the packets from or to the specified hostname

• Display filters are made by:
◇ <protocolname>
▪ Displays any packet using the protocol
- ip
- Displays IP Packets
◇ <protocolname>[.field]
▪ Displays any packet with the specified field present
- ip.addr
- Displays IP packets with a populated source or destination address
◇ <protocolname>[.field] [operand value]
▪ Displays any packet whose protocol field matches the operand and value
- ip.addr == 192.168.12.13
- Displays IP packets with the IP address as source or destination
◇ <protocolname>[.field] AND <protocolname>[.field] [operand value]
▪ You can combine multiple expressions using logical operators
- ip.addr == 192.168.12.13 or arp
- Displays IP packets with the listed IP address or ARP packets
